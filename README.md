# **fs-notes**
Full-Stack Notes app base on project quickstarter for SSR with Express.js and Next.js.

## Libraries used in project:

> **Front-End**
> - React.js with Next.js
> - Material-ui

> **Back-end**
> - Node.js with Express.js
> - NeDB
